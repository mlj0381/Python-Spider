# 58 同城爬虫

### 项目结构

|–项目文件夹

​	|–resutls

​	|–main.py

​	|–requirements.txt

​	|–README.md

`resutls`: 存放爬取数据的文件夹

`main.py`: 主程序

`requirements.txt`: 需要安装的第三方库

`README.md`: 项目描述文件

### 运行

在终端下运行`python main.py`命令
